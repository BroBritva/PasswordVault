# PasswordVault 🔐
Простое WinForms‑приложение для надёжного хранения паролей и кодов восстановления.  
Первая рабочая версия: **v1.0.0**

---

## ✨ Возможности
- Вход по мастер‑паролю
- Хранение паролей и логинов в зашифрованном контейнере (`vault.dat`)
- Шифрование данных с помощью AES‑256
- Простая таблица для добавления и редактирования записей
- Кнопка «Сохранить» для записи изменений

---

## 🛠 Технологии
- C# 9.0+
- .NET 6/7
- Windows Forms
- System.Security.Cryptography (AES, PBKDF2)
- JSON для хранения структурированных данных
